# Analyzing Code with Large Language Models (LLMs)

# Introduction

  • Achievements：Project In Progress
  • Author: First Author
  • Supervising Professor: Hui Liu and Yanjie Jiang
  • Duration：Feb 2024 - Today

# Research Content

  • Search on GitHub for top-level paper codes related to LLM (e.g., GPT-4.0) outside the database.
  • Remove the assert statements from these project codes and feed them to the LLM, allowing the LLM to automatically generate assert statements.
  • By analyzing the assert statements automatically generated by the LLM, the goal is to identify its characteristics and then design software that can automatically complete the test oracle problem. This software will use the LLM interface to perform the automatic generation of assert statements.
