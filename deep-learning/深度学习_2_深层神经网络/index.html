# Tensor——PyTorch 重要元素

# 代码实现

  • w.requires_grad = True ：表示模型需要计算梯度。

  • w : 是一个 tensor
  • forward : 计算y^\hat yy^​
  • loss ：损失函数

  • loss ：计算损失函数
  • l : 损失函数
  • l.backward() : 计算链路上的所有需要的梯度，并且存到变量里面。
  • w.data = w.data - 0.01 * w.grad.data : 因为 w 是一个 tensor ，为了计算具体的数值，需要操作到 w.grad.data
