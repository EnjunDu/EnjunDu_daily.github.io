# 复杂组合逻辑模块设计

# 实验目的

设计一个多功能数字电路模块

  • 输入数据为两个一位十进制数 A 和 B，A 和 B 均为 8421 码表示，其中数据 A 由拨动开关 SW7-SW4 输入（SW7 为 MSB），数据 B 由 SW3-SW0 输入（SW3 为 MSB）
  • 电路的功能包括求和、比较大小、输出最大值和输出最小值四种；分别由如下图所示的按键开关控制，当按键按下时，电路执行相对应的功能并输出结果
  • 输出显示在如下图所示的两个数码管上
  • 求和功能描述：当按下求和键时，对 A 和 B 两个数相加，相加的结果显示在数码管上，注意相加的结果需要进行 BCD 码调整，显示模块必须调用实验一中设计的显示译码电路，当十位为 0 时，需要灭零。
  • 比较大小模块描述：当比较大小功能键按下时，比较 A 和 B 的大小，当 A<B 时显示 01（不灭 0），当 A>B 时显示 10，A=B 时显示 11
  • 输出最大（小）值功能描述：当按下输出最大（小）值功能键时，将 A 和 B 中的最大（小）值显示在数码管上。
  • 灭零：即不亮灯，不显示任何东西的意思

# 实验设计

# 2 位 BCD 加法器模块（2_bcd_adder.v）

# 需求分析

该模块需要对两个 1 位十进制的 BCD 表示数进行相加，并进行 BCD 调整，使结果在数码管上正确显示。

# 设计思路

  • 首先将两个输入数相加，得到临时和。

  • 判断临时和是否大于 9（即是否需要进行 BCD 调整）。

  • 如果临时和大于 9，则需要加 6 进行调整，否则保持原结果。

  • 输出调整后的结果和进位信号。

  • 十位为零的时候要灭零

  • 此时十位为零，调用灭零功能

  • A 为 0101，B 位 0110，此时相加结果为 11
  • 十位为 1

# 比较器模块（2_compare.v）

# 需求分析

该模块需要比较两个 1 位十进制的 BCD 表示数的大小，并输出比较结果。

# 设计思路

  • 比较两个输入数的大小。
  • 如果 A 小于 B，输出 01。
  • 如果 A 大于 B，输出 10。
  • 如果 A 等于 B，输出 11。

# 最大值输出模块 (2_export_max.v)

# 需求分析

该模块需要输出两个 1 位十进制的 BCD 表示数中的最大值。

# 设计思路

  • 比较两个输入数的大小。
  • 如果 A 大于 B，则输出 A。
  • 如果 B 大于或等于 A，则输出 B。

# 设计思路

  • 比较两个输入数的大小。
  • 如果 A 小于 B，则输出 A。
  • 如果 B 小于或等于 A，则输出 B。

# 多路选择器模块（2_mux.v）

# 需求分析

该模块需要根据按键选择不同的功能输出，包括求和、最大值、最小值和比较结果。

# 设计思路

  • 输入部分包括四个按钮信号（分别对应求和、比较、求最大值和求最小值）和四个功能模块的输出。
  • 输出部分包括两个结果输出端口，用于显示选定功能的输出结果。
  • 使用 always 块监视输入信号的变化。
  • 当某个按钮被按下时，选择相应模块的输出并赋值给结果输出端口。
  • 如果没有按钮被按下，则输出两个零。
  • add_output_1 , add_output_2 : 求和功能的两个输出。
  • cmp_output_1 , cmp_output_2 : 比较功能的两个输出。
  • max_output_1 , max_output_2 : 求最大值功能的两个输出。
  • min_output_1 , min_output_2 : 求最小值功能的两个输出。
  • button_a , button_c , button_x , button_n : 分别对应求和、比较、求最大值和求最小值的按钮信号。
  • result_1 , result_2 : 最终输出的结果位，用于连接到显示模块。
  • 通过 always 块监视按钮信号的变化，当某个按钮被按下时，相应的功能输出被选择并赋值给结果端口。
  • 如果没有按钮被按下，输出默认值为两个零，以确保在无输入状态下显示无效值。

# 显示译码器模块（seven_segment_decoder.v）

# 需求分析

该模块需要将 4 位二进制输入（BCD 码）转换为对应的 7 段数码管显示，同时支持一些额外功能，如灭灯、测灯、灭零等。

# 设计思路

  • IB_n ：灭灯信号，优先级最高，低电平有效。
  • LT_n ：测灯信号，优先级次高，低电平有效。
  • RBI_n ：灭零信号输入，低电平有效。
  • RBO_n ：灭零信号输出，低电平有效。
  • P ：最右侧数码管控制信号，用于控制显示状态。
  • 最高优先级的灭灯信号 IB_n ：如果 IB_n 为低电平，则数码管不显示任何内容。
  • 次高优先级的测灯信号 LT_n ：如果 LT_n 为低电平，则所有数码管段都点亮（包括小数点）。
  • 再次的灭零信号 RBI_n ：如果 RBI_n 为低电平且输入数据为 0，则数码管不显示内容，并将 RBO_n 设为低电平。
  • 在以上信号都未触发的情况下，根据输入的 4 位 BCD 码，输出相应的数码管显示编码。
  • 根据输入数据 D 对应的 4 位 BCD 码进行显示转换，并输出相应的 8 位数码管显示编码。

# 顶层模块（complex.v）

# 需求分析

该模块需要整合之前设计的所有功能模块，包括 BCD 加法器、比较器、最大值输出和最小值输出模块，以及多路选择器和数码管显示译码器。顶层模块的职责是协调这些模块的输入输出，实现输入数的求和、比较、最大值、最小值等功能，并将结果显示在数码管上。

# 设计思路

  • 输入：包括两个 4 位 BCD 数 num1 和 num2 ，以及四个功能按钮 B_add , B_cmp , B_max , B_min 。
  • 输出：包括两个 8 位数码管显示 display1 和 display2 ，两个数码管选择信号 select1 和 select2 ，以及两个灭零灯信号 led1 和 led2 。
  • 实例化四个功能模块（BCD 加法器、比较器、最大值输出和最小值输出），将输入数和相应按钮连接到各个模块。
  • 实例化多路选择器模块，将四个功能模块的输出和按钮信号连接到多路选择器。
  • 实例化数码管显示译码器，将多路选择器的输出连接到显示模块，并处理显示信号。
  • 定义中间连线，将各功能模块的输出连接到多路选择器，再将多路选择器的输出连接到数码管显示译码器。
