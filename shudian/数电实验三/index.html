# 数码管扫描点亮电路

# 实验描述

# 实验一个多数码管扫描显示电路

  • 输入 100MHz 时钟，由 EGO1 板通过 FPGA 管脚直接输入

  • 循环点亮四个数码管，四个数码管分别显示 1 2 3 4，显示位置如下图所示，每次只点亮一个数码管，例如，当左侧数码管显示 1 的时候，其余三个数码管均不亮，然后第二个数码管显示 2，同时熄灭左侧显示 1 的数码管，依此类推

  • 时钟频率为 100MHz，即周期为 10ns，要求每个数码管点亮时间保持2262^{26}226 个时钟周期（大约 0.6 秒）

  • 数码管显示译码电路直接调用实验一中所设计的模块

# 改变多数码管显示扫描频率并观察效果

  • 将每个数码管点亮保持时间缩短至2162^{16}216 个时钟周期，观测显示效果，并分析原因。
  • 将每个数码管点亮保持时间缩短至262^{6}26 个时钟周期，观测显示效果，并分析原因。

# 实验目的

  • 了解时序电路的设计流程。
  • 加深学习基于现有电路模块进行复杂电路设计的方法

# 开始实验

# 设计思路

本实验设计的目的是实现一个基于 FPGA 的数码管扫描显示电路。该电路使用 Verilog 硬件描述语言进行设计，主要包括时钟分频器、数码管驱动模块和时序控制模块。

# 时钟分频器

由于 FPGA 的时钟频率通常很高（例如 100MHz），直接用于数码管显示会导致刷新速度过快，人眼无法分辨。因此，我们需要一个时钟分频器，将高频时钟信号分频为较低频率的时钟信号，用于数码管的扫描显示。

# 数码管驱动模块

数码管驱动模块负责控制数码管的点亮和熄灭。通过分时多路复用的方式，我们可以依次点亮每个数码管，从而实现循环显示数字 1, 2, 3, 4 的效果。每个数码管的显示内容通过段选信号控制，显示位置通过数码管选择信号控制。

# 时序控制模块

时序控制模块根据分频后的时钟信号控制数码管的扫描显示。具体来说，通过计数器记录时钟周期数，当计数器达到预设值时，切换到下一个数码管，并重置计数器。这样可以保证每个数码管依次点亮，并保持一定的显示时间。

# 代码展现

# 代码功能说明

# 输入端口

  • clk ：时钟输入信号，接入 FPGA 的时钟源。
  • sw ：控制开关信号，用于启停扫描显示。

# 输出端口

  • select ：数码管选择信号，用于选择当前点亮的数码管。
  • led ：数码管段选信号，用于控制当前显示的数字。

# 内部寄存器

  • count ：计数器，用于计时控制数码管的扫描周期。
  • t ：周期计数器，用于记录当前显示的数码管位置。

# 时钟沿上升触发过程

  • 当 sw 为高电平时，计数器 count 开始计数。
  • 当 count 达到预设值（例如 26'b1111111111_1111111111_111111），表示一个扫描周期结束，重置 count 并递增周期计数器 t 。
  • 根据周期计数器 t 的值，选择对应的数码管并设置段选信号 led 显示相应的数字。
  • 若 sw 为低电平，则重置计数器 count 和周期计数器 t ，停止扫描显示。

通过控制计数器的值，可以方便地调整数码管的扫描频率，从而观察不同频率下的显示效果。这种设计思路清晰，代码简洁易懂，便于调试和扩展。

# 实验周期为2262^{26}226 个时钟周期

在这种情况下，计数器的最大值设置为 26'b1111111111_1111111111_111111 ，即 2^26 - 1 。这种设置下，计数器每经历 2^26 个时钟周期就会溢出，意味着数码管的刷新频率相对较低。

  • 计数器：计数器 count 每个时钟周期加 1，当 count 达到2262^{26}226 时重置为0，并递增周期计数器 t`。
  • 周期计数器：周期计数器 t 依次选择数码管，显示 1, 2, 3, 4。
  • 显示效果：由于刷新频率较低，人眼可以清晰地看到每个数码管依次显示 1, 2, 3, 4。

# 实验周期为2162^{16}216 个时钟周期

在这种情况下，计数器的最大值设置为 16'b1111111111_111111 ，即 2^16 - 1 。这种设置下，计数器每经历 2^16 个时钟周期就会溢出，意味着数码管的刷新频率中等。

  • 计数器：计数器 count 每个时钟周期加 1，当 count 达到 2^16 时重置为 0，并递增周期计数器 t 。
  • 周期计数器：周期计数器 t 依次选择数码管，显示 1, 2, 3, 4。
  • 显示效果：刷新频率较高，但仍在可接受范围内，人眼可以看到每个数码管快速依次显示 1, 2, 3, 4。

# 实验周期为262^{6}26 个时钟周期

在这种情况下，计数器的最大值设置为 6'b111111 ，即 2^6 - 1 。这种设置下，计数器每经历 2^6 个时钟周期就会溢出，意味着数码管的刷新频率非常高。

  • 计数器：计数器 count 每个时钟周期加 1，当 count 达到 2^6 时重置为 0，并递增周期计数器 t 。
  • 周期计数器：周期计数器 t 依次选择数码管，显示 1, 2, 3, 4。
  • 显示效果：刷新频率非常高，人眼可能无法清晰分辨每个数码管显示的数字，可能会看到所有数码管同时亮起的模糊效果。

# 预期结果

  • 较低的刷新频率（2262^{26}226 个时钟周期）会使显示效果清晰，但刷新速度较慢。
  • 中等的刷新频率（2162^{16}216 个时钟周期）使显示效果适中，刷新速度较快且清晰。
  • 较高的刷新频率（262^626 个时钟周期）会使显示效果模糊，刷新速度过快。

# 实验视频及分析

# 2262^{26}226 时钟周期

当时钟周期为 2^26 时，由于计数周期较长，数码管每个数字显示的时间较长，人眼可以清晰地看到每个数码管依次显示 1, 2, 3, 4。这是因为在这样的时间尺度下，每个数码管都有足够的时间单独点亮并显示其数字。

# 原因
  • 刷新频率低：2262^{26}226 个时钟周期的刷新频率较低，数码管依次显示的速度慢，人眼可以分辨每个数码管单独显示的数字。
  • 同步控制：较低的刷新频率可以有效避免 LED 和选择信号的不同步问题，因此不会产生竞争和冒险现象。

# 2162^{16}216 时钟周期

当时钟周期为2162^{16}216 时，由于计数周期缩短，数码管每个数字显示的时间也相应缩短。在这种情况下，虽然数码管依次显示 1, 2, 3, 4，但由于刷新速度较快，人眼可能会看到四个数码管同时显示数字的模糊效果。

# 原因
  • 刷新频率中等：2162^{16}216 个时钟周期的刷新频率中等，数码管依次显示的速度快于2262^{26}226，但仍在一定范围内，肉眼可能看到模糊的同时显示效果。
  • 同步控制：较高的刷新频率可能导致在肉眼看来，四个数码管是同时显示的，但是如果暂停，就可以看到只有一个在显示。所以说较高刷新率，虽然不至于让模块内部产生竞争和冒险，但是对于人眼来说，四个数码管依旧是同时显示的

# 262^{6}26 时钟周期

当时钟周期为262^626 时，由于计数周期极短，数码管每个数字显示的时间非常短。在这种情况下，数码管的刷新速度非常快，人眼无法分辨单独的数字，看到的效果是四个数码管同时显示不同的数字。

# 原因
  • 原因可能是 led 和 select 在电路中没有同步变化，发生冒险与竞争，产生毛刺，导致 1 号管可能显示 '1','2','3' ，2 号管可能显示 '1','2','3','4'，3 号管可能显示 '1','3','4' ，4 号管可能显示 '1','2','3','4' 。由于262^626 个时钟周期极短，导致我们肉眼能同时看到这些数字，从而显示图中的乱码

# 总结

  • 2262^{26}226 个时钟周期：刷新频率低，数码管显示清晰可见，每个数码管依次显示 1, 2, 3, 4。
  • 2162^{16}216 个时钟周期：刷新频率中等，数码管显示模糊，肉眼可能看到四个数码管同时显示。
  • 262^626 个时钟周期：刷新频率高，数码管显示混乱，可能出现不同数码管显示不同数字的现象。
