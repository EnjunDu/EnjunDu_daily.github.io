# 实验一：组合逻辑基础模块

# 实验介绍

# 实验目的

本实验主要基于 EGO1 实现常用组合电路模块，包括编码器 / 译码器 / 显示译码器 / 比较器。实验的主要目的包括：

  • 熟悉基于 FPGA 的组合数字电路设计流程，学习使用 vivado、modelsim 等软件
  • 基于 FPGA 实现常用组合电路模块，加深对相关模块功能的理解

# 实验内容

# 一、8-3 优先编码器设计

  • 8 位输入数据由拨动开关 SW7-SW0 输入，高电平有效（拨动开关向上拨）
  • 输出由发光二极管 LD2 中的 2-0 显示（输出为 1 的时候发光管亮）
  • SW7 的优先级最高，对应输出编码为 111，SW6 优先级次之，对应输出编码为 110，依次类推，SW0 的优先级最低，对应输出编码为 000

# 二、3-8 译码器设计

  • 三位输入数据由拨动开关 SW2-SW0 输入，SW2 为 MSB
  • 输出由发光二极管 LD2 中的 7-0 显示，高电平有效

# 三、四位数据比较器设计

  • 数据 A 由拨动开关 SW7-SW4 输入，SW7 为 MSB
  • 数据 B 由拨动开关 SW3-SW0 输入，SW3 为 MSB
  • 输出由 LD2 的 2-0 输出；当 A>B 时 2 号灯亮，当 A<B 时 0 号灯亮，当 A=B 时 1 号灯亮

# 四、七段译码器

  • 四位输入数据由拨动开关 SW3-SW0 输入，SW3 为 MSB
  • IB_n（灭灯信号）由 SW7 输入，优先级最高，低电平有效
  • LT_n（测灯信号）由 SW6 输入，优先级仅次于 IB_n, 低电平有效
  • RBI_n（灭零信号输入）由 SW5 输入，低电平有效
  • RBO_n（灭零信号输出）由 LD2 的 0 号灯输出，低电平有效，此设计中灭零信号输出不需要与灭灯信号复用管脚
  • 显示输出信号 Ya…Yb 用于控制最右侧数码管的显示

# 8-3 优先编码器设计

# 8-3 优先编码器介绍

使用基础逻辑门电路实现一个 8 线路（输入）至 3 线路（输出）的 8-3 优先编码器。其中，输入信号从 I0 至 I7 为高电平有效，可以视作数字 0 至数字 7 的指示位。8-3 优先权编码器是一个能够将 8 个输入信号进行编码，并根据优先级生成 3 位输出的电路。它可以根据输入信号的优先级，将其中优先级最高的输入转换成对应的三位编码。如果同时有多个输入信号被激活，那么只有优先级最高的信号会被编码器编码。这种编码器通常被用在状况监测、中断控制等场合。

# 3-8 译码器设计

# 3-8 译码器介绍

38 译码器是一种数字逻辑电路，用于将 3 位二进制输入编码为 8 位输出。它的作用是将特定的输入模式映射到相应的输出。在数字电路设计和信息处理中，38 译码器是一个常见且有用的组件，可以帮助实现数据的解码和选择性输出。

3-8 译码器由三个输入引脚和八个输出引脚组成。输入引脚是 A0、A1 和 A2，它们可以是 0 或 1。输出引脚是 YO 到 Y7，它们中的一个会被激活，其它的则处于非激活状态。当输入引脚的二进制值为 000 时，输出引脚 YO 会被激活。当输入引脚的二进制值为 001 时，输出引脚 Y1 会被激活，以此类推，直到输入引脚的二进制值为 111 时，输出引脚 Y7 会被激活。

# 七段译码器

# 七段译码器介绍

七段译码器是一种常见的数字逻辑电路，用于将数字输入转换为对应的七段数码管显示。它通常由四个主要部分组成：

输入端：七段译码器接受数字输入信号，通常是二进制形式，可以是四位、八位或更多位的数字。

译码逻辑：译码逻辑是七段译码器的核心部分，它根据输入信号的状态，将对应的七段数码管的段选引脚控制为高电平或低电平，从而实现数字的显示。

七段数码管：七段数码管是一种常见的数字显示器件，它由七个独立的 LED 段组成，每个段对应一个特定的显示元素，包括水平线段和斜线段，可以显示数字 0 到 9 以及一些字母和符号。

输出端：输出端连接到七段数码管的各个段选引脚，将译码逻辑生成的信号传递给数码管，控制其显示相应的数字或字符。

七段译码器通常用于数字仪表、计数器、计时器、闹钟、温度计等各种电子设备中，提供了一种简单而有效的数字显示解决方案。

# 细节理解

这个实验是关于七段译码器的设计与应用。七段译码器是一种数字逻辑电路，用于将二进制数据输入转换为对应的七段数码管显示。在这个实验中，具体要求如下：

输入数据：四位二进制数据由拨动开关 SW3-SW0 输入，其中 SW3 为最高有效位（Most Significant Bit，MSB）。

灭灯信号（IB_n）：由开关 SW7 输入，优先级最高，低电平有效。这个信号用于指示需要将特定数码管熄灭。

测灯信号（LT_n）：由开关 SW6 输入，优先级次于灭灯信号，低电平有效。类似于灭灯信号，这个信号也用于指示特定数码管的状态。

灭零信号输入（RBI_n）：由开关 SW5 输入，低电平有效。这个信号用于指示需要将数码管中的零段熄灭。

灭零信号输出（RBO_n）：由 LD2 的 0 号灯输出，低电平有效。这个信号通常用于控制其他部分电路中的逻辑，以使零段熄灭。在这个设计中，灭零信号输出不需要与灭灯信号复用管脚。
