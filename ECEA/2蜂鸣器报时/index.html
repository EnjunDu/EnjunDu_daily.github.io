# The buzzer reports the time based on the student ID

# 实验介绍

蜂鸣报时

  • 写一个函数，使用 SysTick 方法，计时 0.25s
  • 使蜂鸣器产生 n 次短鸣 + 1 长鸣，n=mod (学号末位，5)+1
  • 短鸣的时间为 0.25s，长鸣时间为 1s，每次鸣响之间间隔 1s
  • 蜂鸣响起的同时，红色 LED 灯同时亮起
